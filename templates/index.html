User Profile

This web application allows content creators and users to access detailed insights for any YouTube video. To get the dashboard, follow these easy steps:

  • Open the YouTube video you want to analyze.
  • Copy the text after the "=" sign in the video's URL.
  • Enter the copied video ID into the input field below.
